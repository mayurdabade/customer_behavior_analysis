##Customer Behavior Analysis Project
#Overview

This project provides a comprehensive end-to-end analysis of customer behavior, designed to help businesses understand purchasing patterns and optimize marketing strategies. 
Using a multi-tool pipeline of Python (pandas), SQL and Power BI, the workflow focuses on transforming raw transactional data into actionable business intelligence.

The project demonstrates practical skills in:

Data Cleaning and Preprocessing
Exploratory Data Analysis (EDA)
SQL Querying and Data Validation
Data Visualization and Dashboarding
Business Insight Generation

#Dataset
Dataset  contains customer shopping records including:
Products
Product Categories
Purchase Amounts
Payment Methods
Gender and Age Data etc.

The dataset was initially provided in CSV format and processed using Python and SQL Server before visualization in Power BI.

#Tools & Technologies Used   #Tool	Purpose
Python	                      Data cleaning and EDA
Pandas	                      Data manipulation and preprocessing
Jupyter Notebook	            Analysis environment
SQL Server	                  Data storage and querying
SQL	                          Data extraction and validation
Power BI	                    Dashboard creation and visualization

#Project Workflow

#1. Data Loading
Imported CSV dataset into Python using Pandas
Inspected dataset structure, columns, and datatypes

#2. Data Cleaning
Checked for null and duplicate values
Standardized column formats
Converted datatypes where required
Cleaned inconsistent records

#3. Exploratory Data Analysis (EDA)
Performed analysis on:
Sales distribution
Product category performance
Revenue analysis

#4. SQL Analysis
Imported cleaned data into SQL Server
Wrote SQL queries for:
Sales analysis
Customer Type segmentation
Revenue calculations
Category-wise performance

#5. Power BI Dashboard

Created an interactive dashboard to visualize:
Total Customers
Sales Distribution by Category
Product Category Insights
Customer % by Subscription Status
Revenue generation by Age Group

The dashboard allows easy filtering and business-level insight generation

Dashboard Highlights

Key KPIs included:
Total Customers
Average Purchase Amount
Average Review Ratings

Features:
Interactive slicers
Dynamic charts
KPI cards

Report helps in:
Marketing strategies
Customer targeting
Inventory planning
Sales performance tracking

#Conclusion

This project demonstrates an end-to-end data analytics workflow starting from raw CSV data to interactive business report. It showcases practical experience in Python, SQL Server, and Power BI while focusing on deriving actionable business insights from customer shopping behavior data.
